# QuizQuest Arena

Welcome to **QuizQuest Arena**, an RPG battle game where your technical knowledge is your greatest weapon! 

## ⚠️ The Problem

For many developers and students, drilling programming concepts, syntax rules, and computer science trivia can feel dry, repetitive, and unengaging. Traditional flashcards, standard multiple-choice tests, and CLI-based quizzes often lack the immersive excitement needed to encourage daily practice, make learning stick, and prevent burnout.

## 💡 The Solution

An interactive RPG that transforms technical trivia into thrilling back-and-forth battles! 

Players choose a champion from distinct classes (like the nimble *Frontend Sect* or the stalwart *Backend Brawlers*) and face off against a gauntlet of progressively harder adversaries (like the *Null Pointer Disciple* or the *Legacy Code Grandmaster*). 

- **Gamified Combat:** Answer programming trivia correctly to launch devastating attacks against the boss. Answer incorrectly—or run out of time on the countdown clock—and your hero takes damage.
- **Progressive Difficulty:** Questions span Web Development, Software Architecture, General Tech, and more. Bosses are split into Easy, Medium, and Hard tiers with unique health pools and timers.
- **Dynamic Thematic Arenas:** The battleground atmosphere organically shifts entirely according to the difficulty of your opponent.
- **Replayability:** A robust state management system tracks runs, win/loss records, health points, and tracks unique enemy clears.

## 🛠️ Tech Stack

This project is built with a modern, lightning-fast frontend ecosystem:

- **React 18:** Functional components and hooks for seamless UI state management and game-loop handling.
- **TypeScript:** Strict type-safety across player interfaces, enemy classes, and battle events to ensure bug-free combat logic.
- **Tailwind CSS:** Utility-first styling for complex, responsive layouts, glowing auras, and customized interface designs.
- **Framer Motion (`motion/react`):** Powers the core battle engine animations—from fluid UI transitions and swinging weapons to screen-shake impact effects.
- **Vite:** Blazing fast frontend build tooling and local development server.
